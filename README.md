# `repy`

A Rust port of the awesome CLI Ebook Reader `epy` by `wustho`.

![screenshot](https://raw.githubusercontent.com/wustho/epy/master/screenshot.png)

## About

`repy` is a CLI Ebook reader written in Rust, inspired by `epy`. The goal of this project is to port `epy` to Rust, to create a fast and memory-efficient ebook reader for the terminal.

## Features (Planned)

`repy` aims to have feature-parity with `epy`, including:

- Supported formats:
  - Epub (.epub, .epub3)
  - FictionBook (.fb2)
  - Mobi (.mobi)
  - AZW3 (.azw, .azw3)
  - URL
- Reading progress percentage
- Bookmarks
- External dictionary integration
- Inline formats: **bold** and _italic_
- Text-to-Speech
- Double Spread
- Seamless reading
